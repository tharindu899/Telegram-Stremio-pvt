How to add subtitles

From Admin Panel

Select a title, click + Add Subtitle, and enter a Telegram message ID or a public subtitle URL. Stremio will show compact language badges only, like si, en, ta, ar.

Auto-detect via Bot

Send a .srt or .vtt file to the channel — the bot reads the caption first, then falls back to filename, and links automatically.

Kara_2026_Tamil_ESub.srt

TV episodes via Bot

For TV episodes, include season and episode in the caption when sending to Telegram.

[SUB:tt1234567 S01E02 en]
