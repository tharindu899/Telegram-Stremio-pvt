Setup Guide

Install your Stremio add-on

Use an access token manifest URL from the admin panel, then install it in Stremio as a custom add-on.

1. Create token

Open Access Management or Dashboard and create/copy a manifest link.

2. Paste manifest

In Stremio, add a custom add-on and paste the copied manifest URL.

3. Stream

Browse your Telegram-backed catalog and choose a quality to stream.
